# Season27 brand assets

The Season27 mark combines a league-table motif, the number 27, and ranking movement arrows.

- `season27-logo-source.png` is the original generated raster artwork.
- `season27-logo.png` is the transparent production asset.
- `season27-logo-light.png` and `season27-logo-dark.png` are named variants for explicit use on
  contrasting surfaces. The contained dark-green tile gives both variants sufficient contrast.
- `favicon.ico`, `favicon-32.png`, and `apple-touch-icon.png` are browser/device derivatives.

Generation used the built-in image-generation tool with a flat magenta chroma-key background,
followed by local chroma-key removal and Pillow resizing. The prompt requested a crisp,
vector-friendly sports mark with a league table, ranking arrows, implied 27, forest green,
off-white and lime, with no text or copyrighted league symbols.
